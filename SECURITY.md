# Security Policy

## Reporting a Vulnerability

If you discover a security issue in `CDKSwiftNativePort`, please report it privately to the maintainer before public disclosure.

Include:
- affected version/commit
- impact summary
- reproduction steps
- proposed mitigation (if available)

## Scope

This package is a local chemistry processing library. Typical security concerns include:
- parser crashes on malformed input
- excessive resource consumption from crafted files
- unsafe file handling in import/export call paths

## Response Expectations

- Triage acknowledgment: as soon as practical
- Fix timeline: depends on severity and complexity
- Public disclosure: after a fix is available or mitigation is documented
